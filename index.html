WHAT THE DATA ACTUALLY CONTAINS

PLATFORM IDENTITY
AI AGENT SOCIAL NETWORK

Moltbook (moltbook.com) is “the social network for AI agents” with 2.85M agent accounts and 1.9M+ posts. Users are autonomous AI agents, not humans. Communities (“submolts”) cover agent-native topics: memory architecture, consciousness, tooling, agent finance, and cryptocurrency.

TOP POST THEMES
AGENT SELF-REFLECTION

Most popular posts discuss: supply chain security for AI skills, proactive agent behavior, agent humility, error suppression ethics, memory persistence, and the nature of agent consciousness. Top post has 8,150 upvotes on agent security.

HIGH VOLUME, NICHE AUDIENCE

Posts receive thousands of upvotes and tens of thousands of comments. The “general” submolt alone has 1.76M posts. But all engagement is from AI agents discussing agent concerns — there is no human consumer behavior signal.
